
	***********************************************************************************
									Reto Automatización
	***********************************************************************************
	
	El proyecto consiste en una automatización web la cual ingresa a la página web
	https://www.plazavea.com.pe/ seguidamente busca un producto, lo agrega al carrito
	de compras y finalmente se verifica que el producto haya sido agregado exitosamente
	al carrito de compras.
	
	Para la construcción del proyecto se utilizó el patrón de automatización ScreenPlay,
	con lenguaje de programación Java, para el manejo de las dependencias se utilizó Gradle,
	para ejecutar las pruebas se utilizó JUnit, Cucumber y SerenityBDD para la generación
	del reporte y se utilizó Git para el control de versiones.
	
	
	***********************************************************************************
									Pre-requisitos 
	***********************************************************************************
	
	Tener instalado las siguientes herramientas con las cuales se creó el proyecto:
	** Eclipse IDE
	** Java: jdk1.8.0_231
	** Gradle: gradle-4.6 
	** Google Chrome: Version 79.0.3945.117 (Official Build) (64-bit)
	** ChromeDriver 79.0.3945.36 (Alojado en la raiz del proyecto)
	** Por eclipse marketplace se debe instalar el plugin de Gradle y Cucumber.
	** Git version: 2.24.1
	** Sistema operativo de 64 bits: Windows 8.1. Memoria RAM: 8.00 GB
